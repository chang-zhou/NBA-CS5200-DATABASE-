Enviroment: XAMPP win32_5.6.19


1. Setup the enviromen: download and install xampp
2. In xampp control panel, open apache confg(apache httpd.conf), change <document root> route and <directory> to the route of NBA directory
3. Copy and paste files in database directory into ~xampp\mysql\database\
4. Open xampp control panel, start apache and mysql
5. Open internet explorer and enter 127.0.0.1 to view the webpages.
